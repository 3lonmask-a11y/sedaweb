Зачем это

Это не просто «код на макбуке». Это база для сайта, который можно опубликовать.

Проект состоит из обычных файлов: главная страница, стили, небольшой JavaScript и инструкция. Их можно загрузить на GitHub, включить GitHub Pages — и сайт получит публичную ссылку.

Когда захочется поменять текст, цвета или блоки, ты редактируешь файлы и отправляешь изменения в GitHub. Хостинг сам обновляет опубликованную версию.
